# MediaSpider 情报平台缺口开发计划

本文档把当前平台缺口转成可执行开发计划。后续开发必须按阶段推进，每个阶段包含目标、交付物和验收标准，不能跳过。

## 阶段 1：部署与运行基础

状态：已完成基础实现。

目标：让平台可以用标准方式启动、构建、部署和配置。

交付物：

- Dockerfile：后端镜像。
- Dockerfile：前端镜像。
- docker-compose：本地一键启动。
- Nginx 前端代理配置。
- 环境变量模板 `.env.example`。
- 部署说明文档。

验收标准：

- 后端镜像可启动并暴露 `/health`。
- 前端镜像可访问页面，并能通过 `/api` 代理后端。
- 存储目录可挂载到宿主机。
- 默认配置使用 SQLite repository。

## 阶段 2：身份认证与角色权限

状态：进行中，已完成后端认证骨架、前端登录态保护、后端角色权限策略基础和前端权限可见性控制。

目标：从单用户工作台升级为可多人使用的平台。

交付物：

- 用户模型、角色模型、权限策略。
- 登录接口、登出接口、当前用户接口。
- Token 或 Session 认证机制。
- 前端登录页和登录态保护。（已完成基础实现）
- 管理员、分析师、运营、只读审阅者角色。（已完成基础策略）

验收标准：

- 未登录用户不能访问核心 API。（已完成后端保护）
- 不同角色只能看到和操作被允许的功能。（已完成后端基础限制、前端菜单和操作入口控制）
- 前端刷新后能恢复登录态。（已完成基础实现）

## 阶段 3：后台任务调度器与运行队列

状态：进行中，已完成可配置后台调度器基础、同任务并发跳过、扫描串行队列、采集全局并发上限、失败重试记录、运行超时控制，以及 JSON/SQLite 持久化调度历史和重启恢复。

目标：让采集和通知规则能够自动执行，不依赖人工点击。

交付物：

- 后台 scheduler 服务。
- 任务运行队列。（已完成进程内 pending 排队、可配置等待超时，以及手动 API 的 202 后台优先队列提交）
- 并发限制。（已完成同任务 active run 防重、手动/后台扫描串行执行和跨任务全局执行槽）
- 超时控制。（已完成 scheduler run timeout 配置）
- 失败重试策略。（已完成 cron run retry_attempt / max_retries 记录和耗尽跳过）
- 调度运行日志。（已完成审计仓储持久化、最近 50 条重启恢复和前端状态展示）
- 手动扫描与后台扫描统一执行链路。（已完成，统一超时、历史记录和错误响应）
- 调度扫描排队状态。（已完成单执行锁、等待数量 API 和前端状态展示）
- 采集任务容量状态。（已完成 active / queued / max concurrency API 和前端展示）
- 无副作用定时预检。（已完成，仅返回 ready 结果，不创建运行或更新调度时间）
- 服务重启恢复。（已完成单进程模式下 pending 取消、running 失败收口和恢复计数）
- 排队运行取消。（已完成 pending 取消 API、审计记录和任务详情操作）
- 运行中采集终止。（已完成 MediaCrawler 子进程树终止、取消状态收口和任务详情操作）
- 队列优先级。（已完成 critical / high / normal / low 稳定优先队列、同级 FIFO 和前端配置展示）
- 多进程运行租约。（已完成 SQLite 原子抢占、心跳续租、过期恢复和前端状态展示）

验收标准：

- 启动后能按 cron 自动扫描任务。
- 同一任务不会重复并发运行。（已完成）
- 失败任务有明确错误记录和重试次数。（已完成基础实现）
- 调度历史在服务重启后仍可查看。（已完成 JSON/SQLite 模式）

剩余重点：

- 将当前有界后台工作队列升级为独立进程或外部任务队列。
- 自定义 crawler runner 需实现 `cancel(run_id)` 才能支持运行中终止。

## 阶段 4：生产级存储与迁移

状态：进行中，已完成 SQLite 模式配置、JSON/SQLite 迁移文档、容器首次启动自动迁移和存储备份脚本。

目标：让数据存储从开发态 JSON/SQLite 过渡到可生产维护的数据库方案。

交付物：

- Postgres repository 或 ORM 适配。
- 数据库迁移版本管理。
- JSON -> SQLite -> Postgres 迁移路径。
- 备份与恢复说明。
- 数据清理策略。

验收标准：

- Docker compose 可选择 SQLite 或 Postgres。
- 迁移脚本可重复执行并保持数据一致。
- 核心对象具备索引与分页能力。
