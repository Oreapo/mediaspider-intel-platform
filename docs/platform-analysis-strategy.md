# Platform Analysis Strategy

## 总原则

分析体系从“泛内容分析”改为“反黑灰情报分析”，分成三层：

1. 通用风险分析
2. 平台专属风险分析
3. 跨平台关联分析

## 通用风险分析

所有平台都应支持：

- 风险词 / 暗语 / 规避词频率
- 异常活跃度与异常发布时间分布
- 账号批量化内容相似度
- 可疑联系方式、跳转语、导流语提取
- 评论区招募 / 引流 / 灰产话术识别
- 实体共现与重复曝光统计

## 平台专属风险分析

### XHS

- 种草内容中的导流链路识别
- 账号矩阵的内容模板复用检测
- 评论区联系方式、站外引流语识别
- 软广 / 招募 / 兼职类风险话术提取

### DY / KS

- 短视频账号矩阵同步发布识别
- 视频话术与评论话术联动分析
- 导流口播词、字幕词、评论区外链线索提取
- 异常直播带货 / 拉群 / 招募模式识别

### WB

- 热点借势传播中的异常账号簇识别
- 话题扩散链路与引流节点定位
- 舆情操纵、刷量、统一话术扩散识别

### ZHIHU

- 问答内容中的软引流路径识别
- 高重复答案、批量回答账号聚类
- 专业包装型黑灰推广内容识别

### BILI

- 评论区与简介中的导流痕迹识别
- 同模板视频 / 同模板账号矩阵检测
- 社群招募与站外引流路径提取

### TIEBA

- 吧内招募、兼职、灰产教程帖子识别
- 多账号轮番顶帖 / 回帖行为聚类
- 吧群与跨贴吧传播链识别

### XIANYU

- 风险商品价格带与异常波动分析
- 卖家多账号、多商品模板复用检测
- 引流型商品文案与暗语识别
- 地区、发布时间、商品成色的异常模式识别

## 跨平台关联分析

目标不是比较“哪个平台更热”，而是定位“同一风险对象在不同平台的活动轨迹”：

- 同一联系方式 / 昵称 / 文案模板的跨平台复用
- 同一商品描述、图片、价格区间的跨平台映射
- 账号、卖家、内容、商品的传播与导流链路
- 风险实体在不同平台的时间线同步行为

## 输出形态

分析输出建议统一收敛为：

- `signals` 风险信号
- `entities` 风险实体
- `relations` 关系边
- `case_briefs` 案件摘要
- `evidence_packets` 证据包
