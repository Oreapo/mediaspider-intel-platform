# Product Vision

## 产品定义

`MediaSpider Intelligence Platform` 是一个面向**反黑灰情报采集与研判**的多平台工作台。

它的核心不是“多抓一些内容”，而是：

- 从多平台采集公开可见线索
- 把线索整理成可追溯的数据集
- 提取风险信号、行为模式、关系链路
- 形成案件化研判结果和证据输出

## 核心用户

1. 风控 / 反作弊团队
2. 反黑灰治理与安全研究团队
3. 舆情与情报分析团队
4. 商品 / 卖家风险排查团队
5. 法务 / 合规取证协作人员

## 产品目标

### 目标一：统一情报采集入口

统一管理 `xhs / dy / ks / wb / zhihu / bili / tieba / xianyu` 等平台的采集任务，并将采集目标明确为：

- 风险账号
- 风险内容
- 风险商品
- 风险卖家
- 引流线索
- 价格与行为快照

### 目标二：统一线索与数据集

不同平台的原始结果先进入统一数据集，再进一步沉淀为：

- `signal` 风险信号
- `entity` 风险实体
- `case` 案件专题
- `evidence` 证据包

### 目标三：分析驱动而不是文件驱动

结果中心不应只是文件浏览器，而应支持：

- 风险词 / 暗语识别
- 账号 / 卖家 / 商品关联分析
- 行为模式聚类
- 跨平台传播链路分析

### 目标四：案件化输出

最终输出不仅是报表，还应包括：

- 风险摘要
- 证据清单
- 关联对象
- 时间线
- 可导出的专题报告

## 产品边界

不做：

- 平台限制绕过
- 验证码绕过
- 未授权的私密数据获取
- 无边界的大规模持续抓取

只做：

- 用户授权或公开可见数据的低频、可控采集
- 面向反黑灰分析的标准化、关联化、证据化处理
