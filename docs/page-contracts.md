# Page Contracts

## Dashboard Contract

输入：

- 平台状态
- 最近任务运行
- 最新数据集
- 最新风险信号
- 待处理案件

输出：

- 概览卡片
- 风险趋势摘要
- 最近活动流
- 待处理事项

## Tasks Contract

输入：

- 平台清单
- 平台任务模型
- 已保存任务
- 任务运行状态
- 场景模板

输出：

- 任务列表
- 动态任务表单
- 启动 / 停止动作
- 场景化模板选择器

## Datasets Contract

输入：

- 数据集列表
- 数据集元信息
- 抽样数据

输出：

- 数据集表格
- 数据概览
- 字段预览
- 挂接案件入口

## Signals Contract

输入：

- 风险信号列表
- 风险等级
- 命中规则
- 来源数据集 / 任务运行

输出：

- 信号列表
- 风险摘要
- 规则命中详情
- 一键入案动作

## Entities Contract

输入：

- 风险实体列表
- 实体详情
- 关系边

输出：

- 实体卡片 / 列表
- 关系图
- 关联信号与案件摘要

## Cases Contract

输入：

- 案件列表
- 案件详情
- 关联数据集 / 信号 / 实体

输出：

- 案件列表
- 案件摘要
- 时间线
- 研判记录
- 证据包入口

## Analysis Contract

输入：

- 数据集
- 平台分析能力声明
- 分析任务状态

输出：

- 通用风险分析卡片
- 平台专属风险分析结果
- 跨平台关联视图

## Evidence Contract

输入：

- 案件
- 信号
- 实体
- 原始记录引用

输出：

- 证据包清单
- 下载文件
- manifest 摘要
